## Additional Information

To change the notification event/export, you can do so by editing the `config.lua` file.

```lua filename="configs/config.lua" {9-10}
Config = {}

---@param message string
---@param type 'info'|'error'|'warning'|'success'
---@param playerId number?
function Config.showNotification(message, type, playerId)
    local data = { title = 'Information', description = message, type = type }

    if playerId then return TriggerClientEvent('ox_lib:notify', playerId, data) end
    TriggerEvent('ox_lib:notify', data)
end
```
